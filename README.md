# mediadownloader

A Simple Whastapp status and music downloader the app detects the status you viewed and any music execluding the audios recorded which then allow you to download music shared between people easy and effecient.

## Features

- Detects viewed WhatsApp statuses
- Lists shared music files from WhatsApp
- Allows one-tap downloading
- Excludes recorded audio messages
- Clean and user-friendly interface

## Permissions

This app requires the following Android permissions:

- `READ_EXTERNAL_STORAGE`
- `MANAGE_EXTERNAL_STORAGE` (for scoped storage)
- `INTERNET` (if applicable)

Make sure to grant them manually if needed.
